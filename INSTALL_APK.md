## 📲 How to Install on Your Android Phone

### Method 1: USB Cable Transfer (Recommended)

1. **Connect your phone to your computer** using a USB cable

2. **Enable File Transfer mode** on your phone:
   - When you connect, your phone will show a notification
   - Tap it and select "File Transfer" or "MTP" mode

3. **Copy the APK file**:
   - Open File Explorer on your computer
   - Navigate to: `C:\Users\user1\Documents\budget app\BudgetApp\build\app\outputs\flutter-apk\`
   - Copy `app-release.apk` to your phone's `Downloads` folder

4. **Install on your phone**:
   - Open the **Files** or **My Files** app on your phone
   - Navigate to the **Downloads** folder
   - Tap on `app-release.apk`
   - If prompted, **allow installation from unknown sources**:
     - Go to Settings → Security → Unknown Sources (enable)
     - Or Settings → Apps → Special Access → Install Unknown Apps → Files (allow)
   - Tap **Install**
   - Tap **Open** when installation completes

---

## 🔒 Security Note

When installing apps from outside the Google Play Store, Android will warn you about "unknown sources." This is normal for apps you build yourself. The app is safe because you built it from your own code.

To allow installation:
- **Android 8.0+**: Settings → Apps → Special Access → Install Unknown Apps → [Your File Manager] → Allow
- **Android 7.0 and below**: Settings → Security → Unknown Sources → Enable

## ❓ Troubleshooting

### "App not installed" error
- Make sure you have enough storage space (at least 100MB free)
- Try uninstalling any previous version first
- Restart your phone and try again

### "Installation blocked" error
- Go to Settings → Security → Unknown Sources (enable)
- Or Settings → Apps → Special Access → Install Unknown Apps

### App crashes on startup
- Clear app data: Settings → Apps → BudgetFlow → Storage → Clear Data
- Reinstall the app
